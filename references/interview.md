## Core Questions (Always Ask)

### 1. Outcome

```
What outcome do you need?
- Research synthesis / literature review
- Written document / report
- Analysis with recommendations
- Decision / plan
- Other: [describe]
```

### 2. Completion Criteria

```
What does "done" look like?
- Specific deliverable format
- Quality threshold
- Coverage requirements
- Other criteria
```

### 3. Constraints

```
Any constraints to consider?
- Timeline
- Length/format requirements
- Source restrictions
- Audience considerations
- None / flexible
```

## Domain-Specific Questions

### Research Tasks

```
What sources should be included?
- Academic / peer-reviewed
- Industry reports
- News / journalism
- Primary sources
- Specific publications: [list]
```

```
What's the scope boundary?
- Time period
- Geographic focus
- Specific aspects to include/exclude
```

### Writing Tasks

```
What's the target format?
- Report / white paper
- Article / blog post
- Executive summary
- Proposal
- Other: [describe]
```

```
Who's the audience?
- Technical experts
- General business
- Executive level
- Public / general
```

### Analysis Tasks

```
What data/inputs are available?
- Files to analyze
- Sources to gather from
- Existing research
```

```
What decisions does this inform?
- Investment / resource allocation
- Strategy selection
- Risk assessment
- Recommendation for action
```

### Planning Tasks

```
What's the decision context?
- Options already identified?
- Constraints known?
- Stakeholders involved?
- Timeline for decision?
```

## Question Limits

- Max 4-6 questions per interview
- Start with core questions
- Add domain-specific only if unclear
- If uncertain, note in brief.md for clarification during planning
